进程线程

发表于 2017-03-17 | 分类于 linux学习基础

进程的查看。

ps aux 查看当前系统的所有进程

ps aux |grep  名字  grep；过滤作用

将系统进程过滤，只展示有上述名字的进程。

pgrep 只查询进程的PID号

pgrep 想要查看的名字

只显示 想要查看的名字的PID号

firefox 打开火狐浏览器。会占用当前终端

firefox &
不占用当前终端，后台打开火狐浏览器。

jobs 查看当前终端后台运行的程序的信息。有工作好。

#jobs
[1]+ 运行中     firefox &


杀死进程
#kill -9 %1

#jobs
[1] 已杀死    firefox

进程的杀死。

kill 根据进程PID号或者进程名

kill -9 PID号  强制杀死。

pkill -9 进程名

pstree 查看进程树

-p    显示所有的时程，显示结果包含进程名和时进程ID

-u    显示出用户的UID,无论何时，这个UID和进程比较UID参数，这个新的UID将在进程名后显示不同的参数。

-U 　使用utf-8字符集以十进制表示,

-v     显示版本号

dd if=/dev/zero of=/a.txt bs=1G count=1

从zero中复制1个1G到a.txt中：

if 代表输入文件。如果不指定if，默认就会从zero中读取输入。

of 代表输出文件。如果不指定of，默认就会将a.txt作为默认输出。 

bs 将输入与输出设成指定的字节数。 

count 仅读取指定的区块数。

可以测试硬盘的读写速度。
