每次更新博客的步骤

Posted on 2017-03-15 | In 更新博客

常用命令

hexo new”postName”

新建文章

hexo new page”pageName”

新建页面

hexo generate

生成静态页面至public目录

hexo server

开启预览访问端口（默认端口4000，'ctrl + c'关闭server）

hexo deploy

hexo help # 查看帮助

hexo version #查看Hexo的版本
