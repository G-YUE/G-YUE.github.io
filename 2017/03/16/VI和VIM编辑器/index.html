VI和VIM编辑器

发表于 2017-03-16 | 分类于 linux学习基础

VIM编辑器的三种模式

命令行模式：可以输入指定的VI命令来操作正文。
编辑模式：可以输入字符被解释为输入正文。
扩展模式：可以进行一些高级编辑命令，如存盘，退出。

1、编辑器中光标的移动。

$：光标移动到当前行的结尾。
o:光标移动到当前行的开始。

G(shift+g):光标移动到这个文件的最后一行。
nG：n为数字，光标移动到这个文件的第n行。
gg：移动到这个文件的第一行，等同于1G。

H：光标移动到这个屏幕的最上方那一行的第一个字符。
M:光标移动到这个屏幕的中央那一行的第一个字符。
L：光标移动到这个屏幕的最下一行的第一个字符。

2、进入插入模式。

在命令行输入 vim filename,进入的是命令行模式，要想对文件内容修改，需要进入插入模式。

a:进入插入模式，并在光标后添加。

i:进入插入模式，并在光标前添加。

o:进入插入模式，并在当前光标所在行之下，开启新的一行并添加。

3.命令模式下一些功能操作。

如果对自己写入的内容不满意，一个一个的删是否有点麻烦？或者要写一段上面已经有了的话，再从新打一遍？

在命令行模式下有一些简单的操作来实现各种功能：

3.1 常用的删除和复制、粘贴的按键

dd: 删除光标所在的那一行。
ndd： n为数字，删除光标所在的向下n行。

yy： 复制光标所在的那一行。
nyy： 复制光标所在的向下n行。

p： 将已经复制的数据在光标下一行粘贴。
P（shift+p）：将已经复制的数据在光标的上一行粘贴。

3.2 撤销和重做操作。

u：     撤销，复原前一个操作。
ctrl+r：重做上一次操作。

4.扩展模式下常用的命令。

对文件更改完毕，是否保存？怎么退出？这些在扩展模式下。

：w    将文件存入/写入磁盘。
：q    推出vim编辑器，不存盘。
：wq   将文件存入磁盘，并推出。
