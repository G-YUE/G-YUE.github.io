tar：打包

c              归档文件

x              解压缩文件

z              gzip压缩文件

j              bzip2压缩文件

v              显示压缩或解压缩过程 v(view)

f              使用档名

t              列出包的内容

tar cfv test.tar a.txt b.txt c.txt
将a,b,c文件打包成test.tar，并显示详细过程。

如果想要打包某个目录下的所有文件以gzip方式：
tar czf  /test/test.tar.gz /etc/
这种压缩，会把目录名也压缩到包里，解压缩后是以/etc开头的。
cd /etc
tar czf /test/test1/tar/gz *
这种压缩只会压缩/etc下的所有文件，而没有/etc开头。

以gziphi或者bzip2压缩并打包的文件，解压缩的话直接：tar xvf 文件名  可以直接解压到压缩前的状态。

tar -tf test.tar
查看包内的内容

tar xvf test.tar
将包解压到当前目录

tar xvf /test/test.tar -C /bak
将test下的test.tar 解压到bak目录下。

2种常见的压缩和解压缩

gzip:压缩
gunzip:解压缩

打包压缩：
tar cvzf test.tar.gz a.txt b.txt c.txt
解压打包：
tar xvf /test/test.tar.gz -C /bak

bzip2:压缩
bunzip2:解压缩

打包压缩：
tar cvjf test.tar.bz2 a.txt b.txt c.txt
解压：
tar xvf /test/test.tar.bz2 -C /bak

有上面看出2个压缩方式似乎一模一样。但是我们看一下2种方式压缩后的效果：

下面我们建立2个文本文档：1.txt和2.txt。两个文档大小都一样。
我们分别用gzip压缩1.txt，用bzip2压缩2.txt。然后查看2个压缩后的文件。

从结果中我们可以看出2种方式的差距了。

gzip：  压缩速度快，压缩率小。
bzip2： 压缩速度慢，压缩率高。
