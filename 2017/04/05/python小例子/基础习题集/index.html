3.简述 ascii、unicode、utf-­‐8、gbk 的关系

Ascii是最早的一套编码，后来国人扩展了Ascii表，制出了CB2312编码表，后来又扩展到了GBK编码表。后来又有了GB18030。后来国际化标准组织为了规范化编码标准，研究出了unicode编码。经过一系列进化，发展到了可变长的utf-8编码
